CSS: design visual para a Web

Postado por Dalton Serey em 02 de dezembro 2021

Web Apps modernos dependem cada vez mais de frontends funcionais, eficientes e, não menos importante, atrativos. É nesse contexto que o trio de linguagens da Web brilha. HTML é a linguagem veículo dos conteúdos e da estrutura. JavaScript a linguagem do comportamento, da lógica da interatividade. E, por fim, cabe a CSS abordar todos os aspectos relativos à apresentação do conteúdo e, portanto, parte significativa daquilo que definimos como a atratividade do aplicativo.

Neste post, focamos na importância de se aprender e usar CSS corretamente, como forma de maximizar as chances de produzirmos web apps de sucesso.

CSS

CSS (Cascading Style Sheets) é uma linguagem de folhas de estilo. Tal como HTML, é uma linguagem declarativa que permite especificar regras de apresentação e estilização de elementos HTML. É com base nessas regras que o Rendering Engine do browser irá gerar os detalhes das representações visuais dos elementos. As regras incluem informações sobre distribuição espacial (layout), tamanhos, fontes, cores, etc.

Infelizmente, há certo preconceito por parte de alguns engenheiros de software com relação a CSS. Os motivos são vários: a linguagem deveria ser ferramenta de trabalho do designer; a linguagem é difícil de controlar e, em geral, requer conhecer dezenas de propriedades e uma sintaxe um pouco criptográfica para selecionar elementos; a linguagem não tem (ou tem pouco) suporte para decompor e estruturar código grande de folhas de estilo ao estilo das linguagens de programação; etc. Por um lado, estas afirmações têm base. Por outro, esses são justamente os motivos pelos quais é necessário estudar e dominar a linguagem ao invés de insistir em abordagens de codificação baseadas em tentativa e erro (na verdade, esses motivos somados ao fato de que a linguagem é o padrão de fato para design de páginas e aplicativos web.

Regras CSS

Uma regra CSS é extremamente simples. Cada regra especifica um conjunto de elementos do HTML aos quais a regra se destina e um conjunto de declarações que especificam os valores de várias propriedades visuais e de apresentação que o browser deve usar ao renderizar e pintar o elemento. Essas duas partes são conhecidas:

  1. um seletor; e
  2. um bloco de declarações.

Várias propriedades dos mais diversos elementos podem ter seus valores ajustados com regras CSS. Abaixo algumas das propriedades mais importantes que os browsers usam ao renderizar texto:

  • font-family
  • font-size
  • font-weight
  • font-style
  • text-align
